# Agent Registration System

## Objective
Provide a robust system for agents to register, validate, and onboard into Royal Mix Global.

## Key Features
- Online registration (address, phone, ID, photo)
- Road / village assignment
- Agreement & compliance tracking
- Automated approval & notifications

## Agent Role
- Register and validate profile
- Select operational zone
- Accept agent agreement digitally

## Metrics & KPIs
- Total registered agents
- Approval rate per region
- Average registration time
- Compliance score
