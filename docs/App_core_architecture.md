# Royal Mix Global – App Core Architecture

## Purpose
The app is a window into the Royal Mix Global system.
It connects physical reality to digital intelligence.

---

## Core User Roles
- Founder / Admin
- Agent
- Distributor
- Partner (NGO / Company / Government)

---

## Core Screens (MVP)

### 1. Agent Dashboard
- Daily sales input
- Feedback (taste / demand)
- Zone performance
- Level progression

Purpose: Capture real market data.

---

### 2. Product Engine Screen
- Active products (Royal Mix Choco, others)
- Sizes & variants
- Test vs Scale status

Purpose: Control product evolution.

---

### 3. Data Intelligence Screen
- Heat maps (zones)
- Demand signals
- Growth trends
- AI recommendations

Purpose: Decision-making brain.

---

### 4. Financial Flow Screen
- Daily agent totals
- Weekly summaries
- Monthly performance

Purpose: From street cash to company logic.

---

### 5. Ecosystem Expansion Screen
- New product proposals
- Agent upgrades
- Region expansion readiness

Purpose: Controlled scaling.

---

## Architecture Logic
Physical World → Agents → App → Data → AI → Decisions → Physical World

The app does not replace reality.
It amplifies it.

---

## Why This App Is Different
- Uses real consumption data
- Trains AI on physical markets
- Grows with human networks
- Is multi-product by design

Most apps are digital.
Royal Mix Global is **phygital**.
